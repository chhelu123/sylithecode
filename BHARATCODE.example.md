# BharatCode — Project Config

This file is automatically injected into BharatCode's system prompt.
Create this file in any project to give BharatCode custom instructions.

## Example Project Config

```markdown
# My Project — BharatCode Config

## Tech Stack
Python 3.11 / FastAPI / PostgreSQL / React 18

## Coding Standards
- Use Black for formatting
- Type hints everywhere
- Async functions preferred

## Indian Integrations
- Razorpay v2 for payments
- UIDAI sandbox for Aadhaar OTP
- GST rates: 18% for services

## Notes for BharatCode
- Never commit .env files
- Run `python -m pytest` after every fix
- Use `alembic` for database migrations
```
